# Day1 

使用 CLI 建立專案



### 步驟

安裝 Vue cli

```shell
$ sudo npm install -g @vue/cli
```

安裝完成後，確認版本：

```shell
$ vue -V
```

建立專案

```shell
$ vue create vue-demo
```

建立的時候，選擇要使用 Vue2 或者 Vue3，跑完就安裝完成



執行專案

```shell
$ cd vue-demo/
$ npm run serve
```

就可以在瀏覽器開啟：http://localhost:8080/ 來觀看畫面



也可以使用

```shell
$ vue ui
```

的 GUI 工具來管理專案
